# Electrofishing Fish

A tibble of the fish caught by electrofishing.

## Usage

``` r
effish
```

## Format

A tibble with the columns:

- EFSite:

  Site names

- DateEFVisit:

  Date of visit

- EFPass:

  Electrofishing pass number

- FishNumber:

  Fish number

- Species:

  Species code

- ForkLength:

  Fish fork length (mm)

- BodyWeigth:

  Fish body weight (g)
