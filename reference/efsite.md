# Electrofishing Site Data

A tibble of the different backpack electrofishing sites.

## Usage

``` r
efsite
```

## Format

A tibble with the columns:

- EFSite:

  Electrofishing site name

- Creek:

  Creek name

- SiteLength:

  Site length (m)

- Elevation:

  Site elevation (masl)

- DominantSubstrate:

  Dominant substrate

- geometry:

  Site upstream spatial coordinates.
