# Periphyton Data

A tibble of the periphyton time series Chlorophyll-A concentrations.

## Usage

``` r
periphyton
```

## Format

A tibble with the columns:

- BioSite:

  Site name

- DatePeriphyton:

  Sample date

- Series:

  Series number

- SampleID:

  Unique sample identifier

- Chla:

  Chlorophyll-A concentration
